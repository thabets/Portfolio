This is an example of Note Taker App

Note Taker

  • Category: Web design
  • Client: University of Toronto
  • Project date: 05 September, 2021
  • Project URL: https://notetakersam.herokuapp.com/

Note Taker is an application as named with the function of a Note Taker, where the user is able to add title as well as notes associated with the title. The user then can proceed to save those notes or is able to also delete any notes that appear to them from the list of notes that were previously entered. The images to the left show a simple view of the application. Feel free to click on the above URL and test it out for yourself.
